box-sizing :content-box :- makes the box size based on the content
ex : if defined width is 400 px, then the box is much more than 400 px 

box-sizing :border-box :- makes the box's elements size based on the box
ex : if defined width is 400px, then the box's width is 400 px, and the content is less than 400px
